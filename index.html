Politica de Privacidad - Siempre Fresco

Ultima actualizacion: 14/04/2026

Esta Politica de Privacidad explica como Siempre Fresco accede, usa, almacena y protege la informacion relacionada con el uso de la aplicacion movil.

1. Responsable y contacto

Aplicacion: Siempre Fresco

Contacto de privacidad y soporte: vikena.smart@gmail.com

2. Datos que procesa la app

Siempre Fresco esta disenada para funcionar principalmente con datos almacenados en el dispositivo del usuario. La app puede procesar las siguientes categorias:

  • Datos ingresados por el usuario: nombre del producto, marca, descripcion, categoria, ubicacion, cantidad, precio, codigo de barras o QR y fecha de vencimiento.
  • Datos tecnicos locales: idioma seleccionado, hora de recordatorio, estado de notificaciones y rutas de carpetas elegidas por el usuario para respaldos.
  • Datos derivados del uso de la camara: lectura de codigos de barras o codigos QR cuando el usuario activa voluntariamente la funcion de escaneo.
  • Archivos de respaldo: archivos JSON exportados o restaurados por el usuario con la informacion de inventario y productos recordados.

3. Permisos y finalidad

  • Camara: se usa solamente para escanear codigos de barras y QR cuando el usuario lo solicita.
  • Notificaciones: se usan para enviar recordatorios locales sobre productos proximos a vencer, solo si el usuario activa la funcion.
  • Acceso a archivos seleccionados por el usuario: se usa para importar o exportar respaldos manualmente mediante el selector de archivos o carpeta.
  • Internet: se usa para abrir enlaces legales externos y, si el usuario utiliza funciones que dependan de servicios externos configurados por el desarrollador, para realizar solicitudes HTTPS.

4. Como se usan los datos

  • Mostrar y organizar el inventario de alimentos.
  • Detectar productos proximos a vencer y generar recordatorios locales.
  • Recordar informacion asociada a codigos escaneados para agilizar futuras capturas.
  • Crear y restaurar respaldos locales solicitados por el usuario.
  • Aplicar preferencias de idioma y configuracion dentro de la app.

5. Almacenamiento y retencion

La informacion principal del inventario y configuraciones se almacena localmente en el dispositivo del usuario. Los respaldos automaticos y manuales permanecen disponibles hasta que el usuario los elimine, desinstale la app o los sobrescriba.

6. Comparticion de datos

Siempre Fresco no vende datos personales. La app no comparte automaticamente la informacion del inventario con terceros con fines publicitarios.

Puede existir transferencia de datos solo en estos casos:

  • Cuando el usuario exporta o comparte voluntariamente un archivo de respaldo.
  • Cuando el usuario abre un enlace externo.
  • Cuando una funcion opcional configurada por el desarrollador usa un servicio externo de traduccion o politicas de version mediante HTTPS.

7. Servicios de terceros

La aplicacion puede integrar bibliotecas o servicios tecnicos necesarios para funcionalidades locales del dispositivo, como notificaciones, seleccion de archivos, compartir archivos o escaneo de codigos. Si se configura un servicio externo de traduccion o de politicas de version, el texto enviado a ese servicio sera procesado segun las condiciones y politicas del proveedor correspondiente.

8. Seguridad

Se aplican medidas razonables para proteger la informacion manejada por la app. Sin embargo, ningun metodo de almacenamiento o transmision es completamente infalible y el usuario debe proteger el acceso a su dispositivo y a sus respaldos.

9. Menores de edad

La app no esta dirigida especificamente a menores ni busca recopilar deliberadamente datos personales de menores de edad.

10. Control del usuario

  • El usuario puede editar o eliminar elementos del inventario desde la app.
  • El usuario puede desactivar notificaciones desde la configuracion del dispositivo o desde la app.
  • El usuario puede eliminar archivos de respaldo creados manual o automaticamente.
  • El usuario puede desinstalar la app para eliminar el almacenamiento local administrado por la aplicacion, salvo archivos exportados fuera del area privada de la app.

11. Cambios en esta politica

Esta Politica de Privacidad puede actualizarse para reflejar cambios legales, tecnicos o funcionales. La version vigente sera la publicada en esta pagina.
